# The Detectorist

A 3D first-person metal detecting adventure. Single-file browser game (Three.js).

- Play: https://lawquarter.github.io/detectorist/
- Source modules: `data.js` (items/detectors/sites), `world.js` (3D world), `game.js` (game loop), `ui.html`, `style.css`
- Rebuild `index.html` after editing modules: `./build.sh`
